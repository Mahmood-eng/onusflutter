📂 المشروع الأول: الأساسيات والـ Stack
المجلد: onus1

شرح حالة المكدس وتأثره عند كل عملية تنقل.

📂 المشروع الثاني: تبادل البيانات (Data Flow)
المجلد: onus2
الفكرة: إدارة البيانات بين الشاشات في الاتجاهين (رايح جاي).
التنفيذ: * Push + Data:
تمرير اسم المنتج من القائمة إلى صفحة التفاصيل عبر الـ Constructor.

Pop + Result: إرجاع "نتيجة" عند العودة (مثل إضافة المنتج للمف .


Feedback: عرض النتيجة المستلمة في الشاشة الرئيسية باستخدام SnackBar لإشعار المستخدم.
